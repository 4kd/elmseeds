About Elmseeds

Elmseeds is produced by Erik Person. New episodes are released on Thursdays with minor breaks for holidays. Each episode intends to teach a feature of Elm, a particualr package, or a unique way to use the language.

If you have an episode suggestion, please contact me via Twitter.
